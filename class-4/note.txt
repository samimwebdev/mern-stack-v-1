REST API
=========
GET - Getting all tasks from server (/tasks)
GET - Getting a single task form Server (/tasks/:id)
POST - Adding a new Task in Server(/tasks)
PUT - Update a single task in Server (/tasks/:id)
DELETE - DELETE a single task from Server (/tasks/:id)
